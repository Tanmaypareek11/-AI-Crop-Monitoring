# Wheat Disease Classification Model

## Setup
1. Install dependencies:
   pip install -r requirements.txt

2. Organize your dataset folders as:
   - train/
   - valid/
   - test/
   Each containing class-specific subfolders.

## Training
Run:
    python scripts/train.py

## Evaluation
Run:
    python scripts/evaluate.py

## Prediction
Run:
    python scripts/predict.py path_to_image.jpg
