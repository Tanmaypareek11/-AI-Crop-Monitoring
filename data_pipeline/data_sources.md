# Data Sources and Data Types

## Data Collection Sources

The data for the AI Crop Monitoring system was collected from multiple real-world sources:

1. **Satellite Multispectral Imagery**
   - Used for extracting vegetation health indicators.
   - Provided large-scale crop monitoring coverage.

2. **Ground Sensor Networks (IoT)**
   - Soil moisture sensors
   - Soil pH sensors
   - Temperature and humidity sensors

3. **Weather Data Feeds**
   - Historical and real-time weather information
   - Rainfall, temperature trends, humidity levels

4. **Drone Surveys**
   - High-resolution crop imagery
   - Disease detection and localized crop stress monitoring

---

## Extracted Data Types

The following structured features were derived from the collected data:

- Vegetation Indices:
  - NDVI (Normalized Difference Vegetation Index)
  - EVI (Enhanced Vegetation Index)

- Soil Parameters:
  - Moisture content
  - pH levels

- Environmental Factors:
  - Temperature readings
  - Humidity readings

- Pest Sensor History:
  - Pest activity frequency
  - Infestation patterns
